Benchmarking existing feature matching methods for localization

This is the official benchmarking for UAV2-20 visual localization using feature matching. The benchmark will measure the following metrics:

1. Execution time: Runtime of the method with breakdown of each components
2. Localization accuracy: Each satellite image are geotagged, thereby we can establish the groundtruth location of every pixel within the image. The predicted correspondences between uav image's pixel and satellite image's pixel can be use to assigned a location to the pixel in uav image. 
3. Pose Estimation: This will based on preexisting dataset since the project's data is not labelled. This measurement will only for rough estimation of how each methods perform. This is not intrumental to the project in anyway so it is not likely to be calculated. I just write it here if I cave out time to do more experiments.

We need to visualize the results and create an evaluation system that facilitate thorough and convenient measurement. How to define "convenient" is an open question for now.
